# Student Record Manager 📋

A Python mini project built using Lists and Tuples to manage student records.

## Features

* Add Student Records
* Store Data using Tuples
* Manage Multiple Students using Lists
* Display Student Details

## Concepts Practiced

* Lists
* Tuples
* Loops
* User Input
* Basic Data Management
